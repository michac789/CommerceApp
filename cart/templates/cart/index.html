API Temporary Mini Documentation
/api/bookmark/(id)
GET:
gets all of your bookmarked items (item_id does not matter)
returns serialized version of that item to be put on DOM
PUT:
bookmark that item
DELETE:
remove bookmark from that item

/api/cart/(id)
GET:
gets all of your items on cart (item_id does not matter)
returns serialized version of that item to be put on DOM
PUT:
add item on cart (cannot add your on sale item or closed item)
DELETE:
remove item from cart
